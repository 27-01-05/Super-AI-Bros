# Super Marco Bros (Original Retro-Inspired Platformer) I can’t help make a near-identical copy of World 1-1 with the same music, sounds, exact layout, or assets from Super Mario Bros. But I *can* help you make an original retro-inspired platformer with: * Pixel graphics * A plumber-style hero * Goomba-like enemies * Coin collecting * Brick blocks * Retro beep music * NES-style gameplay feel * A custom level inspired by classic platformers Here’s a simple starter version using HTML + JavaScript. Save this as `index.html` and open it in your browser. ```html Super Marco Bros ``` ## Controls * Arrow keys = Move * Space = Jump ## Ideas To Add Later * Flagpole ending * Underground level * More enemies * Power-up mushrooms * Fireball attacks * Pixel title screen * Retro beep soundtrack * Animated bricks * Secret pipes
